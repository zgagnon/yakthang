# bin/

This directory previously held local build artifacts. Binaries are now installed to standard locations by `just install`:

| Binary | Install location |
|--------|-----------------|
| `yx` | `~/.local/bin/yx` |
| `yak-box` | `~/.local/bin/yak-box` |
| `yak-map.wasm` | `~/.local/share/zellij/plugins/yak-map.wasm` |
